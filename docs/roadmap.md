# Roadmap and Progress

最終更新: 2026-01-18

このドキュメントは、Simple Docking Dashboard プロジェクトの開発ロードマップと実装進捗を管理します。

## 🎯 プロダクト方針（化学者向け）

- **想定ユーザー**: 合成化学者。計算/生物の専門知識がなくても使える前提。
- **価値仮説**: Ketcher で描いて Run を押すだけで、既存化合物と比較した「候補性の示唆」が得られる。
- **ユースケース**: 日々の合成化合物の物質特許資料に、比較結果を簡潔に提示できる。
- **位置づけ**: 高精度な予測ではなく、比較とコミュニケーションの起点。
- **期待効果**: 特許資料を見た計算化学者/バイオ系人材との協業につながる。
- **成功指標**: 初見でも迷わず完走し、数分で示唆とレポートが得られる。
